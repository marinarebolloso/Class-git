Secuencia de escape
\'= Mostrar una comilla simple dentro de una cadena en la salida de información
\" = Mostrar una doble comilla simple dentro de una cadena en la salida de información
\\ = Barra invertida, diagonal inversa nos permite mostrar una back slash
\n = salto de línea
\r = Retorno
\t = tabulación
\b = retroceso
\f = salto de página

Operador de aumento 
++
Ejemplo: Let z = ++a; //PreIncremento (el operador antes de la variable, a la variable a (ej 1 le suma un número más))
Ejemplo: z = a++; //PostIncremento (la variable z va a recibir el valor que tiene la variable a ej a = 2 z = 2// Para el próximo uso a va a valer 3 pero z sigue con 2)

Operador decremento 
--
Ejemplo: Let z = --a; //PreDecremento
Ejemplo: z = a--; //PostDecremento

Operador Distinto, Diferente (Boolean)
!= Comprueba valor sin revisar el tipo
!== Comprueba valor y revisa el tipo
